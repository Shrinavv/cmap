# CMAP

CMAP is a consent management platform that has 2 major components : a dashboard that makes it possible for you to view all the cookies 
avaiable in your browser, and an extension that gives the ability to view the cookies live getting updated and the feature to delete them anytime.

---

# Setup
```powershell
git clone https://github.com/Shrinavv/cmap.git
cd cmap/backend
npm install
npm run dev
cd ../dashboard
npm install
npm run dev
```
